About Flipit Map Creator

The goal of this project is to provide a way to create and modify Flipit map files. It is basically a companion to the game, Flipit. While Flipit is still in development stage, I'll continue to improve this project's functionality with goals and milestone planned (See readme, milestone and issues section in the project github page). This is a personal side project of mine, an opportunity to improve my skills and knowledge.

Features

  • Create and replace tiles on click
  • Can create map with size up to 20 by 20 tiles.
  • Automatically saves a preview image file of the map.
  • Display a log of events to monitor action history

Bug reports and contributions

I try my best to squash as many bugs as possible, but occasionally they pop out of nowhere. if you like to notify me or just send a simple feedback, please do create an issue on project's Github page.
