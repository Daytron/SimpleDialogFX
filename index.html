SimpleDialogFX

About SimpleDialogFX

I created a very simple, minimalist modal dialogs for JavaFX, since it doesn't have one (crazy I know). It is lightweight and easy to use.

Usage

Introduction on its usage can be seen on the project's readme. See Javadoc for more information.
